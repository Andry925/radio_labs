## Мета роботи

Освоїти створення проєкту в STM32CubeMX для STM32F401.
Налаштувати GPIO через HAL (різні порти та виводи).
Реалізувати формування цифрового сигналу через software delay (polling loop).
Дослідити вплив частоти SYSCLK на часові параметри сигналу.

---

## Завдання (Варіант 6)

PC7, Push-Pull, меандр 5 Гц, SYSCLK = 16 Mhz
